## Struktura projektu

- `src/zadania` - rozwiązania zadań z zestawu 4.
- `src/example` - przykładowe zadania przesłane przez grupę.

## Zarządzanie pakietami

W profesjonalnym programowaniu kod zawsze dzieli się na 
pakiety (np. według funkcjonalności: controller, service, repository). 
Trzymanie wszystkiego w src (tzw. default package) jest uważane za złą praktykę.

Nazewnictwo pakietów `Reverse Domain Name Notation`

Pakiety są domyślnie wyświetlane w jednej linii (z kropkami), Aby wyświetlić
strukturę drzewiastą, nalezy wyłączyć opcję `Compact Middle Packages`
